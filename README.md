# SlideCraft

SlideCraft is a beautifully designed Flutter application that allows users to create stunning cinematic slideshows with ease. Embodying "The Quiet Ceremony" design system, it features a Japandi-inspired, minimalist, and highly tactile user interface.

## Features
- **Cinematic Video Rendering**: Powered by FFmpeg, create seamless video slideshows from your images directly on your device.
- **Japandi UI Design**: A meticulously crafted design system focusing on bilateral asymmetry, tonal layering, and an elegant editorial typography experience.
- **Media Management**: Easily select and manage media using integrated file and image pickers.
- **Instant Preview**: View your cinematic creations with the built-in video player.

## Getting Started

### Prerequisites
- [Flutter](https://docs.flutter.dev/get-started/install) (sdk: ^3.5.3)
- Dart SDK

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sh9336/SlideCraft.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SlideCraft
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the application:
   ```bash
   flutter run
   ```

## Design North Star: "The Quiet Ceremony"
SlideCraft strictly follows an intentional fusion of Japanese wabi-sabi and Scandinavian functionalism, utilizing:
- **Colors**: Tactile neutrals mimicking handmade washi paper, with `surface` (#fffbff) as the canvas.
- **Typography**: A high-contrast scale featuring Noto Serif for elegant display and Plus Jakarta Sans for functional layout.
- **Depth**: Minimalist three-dimensionality achieved through tonal layering rather than standard harsh shadows.
